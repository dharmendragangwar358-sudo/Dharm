# 🎮 Random Python Game

प्रथम वर्ष के BTech CSE छात्र द्वारा बनाया गया एक मजेदार और इंटरैक्टिव पायथन गेम। यह प्रोजेक्ट पायथन के `random` मॉड्यूल का उपयोग करके बनाया गया है।

## 🚀 फीचर्स (Features)
* सरल और समझने में आसान यूजर इंटरफेस (CLI-based)
* पायथन के `random` मॉड्यूल का बेहतरीन उपयोग
* शुरुआती प्रोग्रामर्स के लिए सीखने का अच्छा जरिया

## 🛠️ कैसे चलाएं (How to Run)
इस गेम को अपने कंप्यूटर पर चलाने के लिए नीचे दिए गए स्टेप्स को फॉलो करें:

1. इस रिपोजिटरी को क्लोन करें या ज़िप फ़ाइल डाउनलोड करें।
2. अपने सिस्टम में टर्मिनल/कमांड प्रॉम्ट खोलें।
3. फाइल वाले फोल्डर में जाकर नीचे दी गई कमांड चलाएं:

```bash
python "import random_game.py"
```
